Guide

検知チューニングの説明

RTSP / USB切替

RTSP をオンにすると USB / CSI の代わりに RTSP URL を読みます。別端末の配信を使ったデバッグに向いています。

最小輪郭 / 最大輪郭

小さすぎるノイズや大きすぎる誤検知を除外します。ミニカーでは最小輪郭を下げると拾いやすくなります。

二値化

背景差分後の白黒化の強さです。下げると拾いやすく、上げるとノイズを抑えやすくなります。

背景履歴 / 背景しきい値

背景モデルの安定度を決めます。履歴を大きくすると安定しやすく、しきい値を上げると変化に鈍くなります。

Blur / Morphology

ブラーは細かなノイズをなだらかにし、Morphology は小さなゴミ除去と穴埋めをします。重いときはこの2つを切ると効きやすいです。

Open / Dilate

Open は小さなゴミを削り、Dilate は検知領域を少し太らせます。やりすぎると形が膨らみます。

追跡距離 / 見失い許容

前フレームとの対応づけ条件です。速い物体は距離を少し広げると追跡しやすくなります。

再検知クールダウン

一度検知したあと、同じ track を何秒抑えるかの設定です。無視するか、小さく表示するかを選べます。

青床除外 HSV

カラーピッカーで床色を選び、その周辺色を HSV で除外します。数値欄は微調整用です。

Line Crossing

Line A と Line B を通過した時間差で速度を出します。distance_m には2本の実距離を入れてください。

ppm / px/s

Scale 未設定の間は px/s 表示です。実速度にするには Scale で2点と既知距離を設定します。
