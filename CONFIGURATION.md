# 設定方法

ブラウザで `http://<IP>:5000` にアクセスし、以下の設定を行います。

## カメラ設定

- カメラ種別: `usb` / `csi`
- デバイス番号
- 解像度
- FPS

## ROI設定

- 多角形座標を JSON 配列で入力します
- `enabled` をオンにすると、ROI外の動体を無視します

## Perspective補正

- コース領域の4点を `[[x1,y1],[x2,y2],[x3,y3],[x4,y4]]` 形式で入力します
- 保存時に homography 行列を自動計算して `config.json` に保存します

## スケール設定

- 基準になる2点を入力します
- 実距離をメートルで入力すると `ppm` が自動計算されます

## 反映タイミング

- 設定保存後は `config.json` が更新されます
- `speed_estimator.py` や `/stream` を再起動すると新しい設定で動作します
